\chapter{Lejos Statemachine}
\emph{Lejos Statemachine development toolkit} è una utility per \emph{Eclipse},
che consente di progettare graficamente una macchina a stati e implementarla in un
brick Lego\textregistered~NXT. Il plugin genera codice Lejos, pertanto consente
al programmatore di inserire normale statement Java (comprendendo, chiaramente,
le API di Lejos) all'interno del diagramma a stati.\\
Il manuale ufficiale (in lingua tedesca) del plugin è disponibile al seguente
indirizzo: \url{http://fermat.nordakademie.de/update/Beschreibung.pdf};
l'autore ne ha prodotto anche un breve riassunto in inglese che si può trovare
qui: \url{http://lejos.sourceforge.net/forum/viewtopic.php?t=675}.
